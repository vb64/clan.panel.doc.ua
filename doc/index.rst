Добро пожаловать в документацию клан-панели WOT!
================================================

Сайт клан-панель призначений для адміністраторів кланів (командир, заступники, плюс настроюваний список посад) онлайн-гри WorldOfTanks.
Сайт допомагає у вирішенні наступних проблем:

* Відстеження кланових подій
* Облік техніки, наявної в клані
* Відстеження поточної ігрової активності учасників клану
* Формування кланових рот і підбір для них особового складу
* Облік відвідування кланових заходів учасниками клану
* Статистика вільного онлайну клану
* Управління та захист для укріпрайону клану
* Пошук атак УР
	
Клан-панель обслуживает следующие кластеры World Of Tanks:

* `Росія`_
* `Європа`_
* `Північна Америка`_
* `Південно-східна Азія`_
